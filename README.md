# data warehouse and analytics project
this project demonstrates data warehousing and analytics solution, from builsing the datawarehouse to generating analytical insights.

# Project Requirements
Buiding the Data Warehouse (Data engineering)

objective : Develop a modern datawarehouse  using SQL Server to consolidate sales data, enabling analytical reporting nd decision making.

Specification:
1. data sources : import data from two source CRM and ERP (provided as CSV).
2. Data quality : cleaning and standardizing data.
3. Integration : combined both sources into single user-friendly data model.
4. Scope : focus on latest dataset only, historization of data is not required.
5. documentaion : provide clear documentation of data model to support both stakeholders and team.
